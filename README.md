# AWS S3 Static Website Hosting

This project demonstrates how to host a static HTML website using **Amazon S3 Static Website Hosting** without using any web server.

## 🚀 Services Used

- Amazon S3
- Static Website Hosting
- Bucket Policy

## 📁 Project Files

- `index.html`
- `error.html`

## 📌 Steps

1. Created an S3 bucket.
2. Disabled **Block all public access**.
3. Uploaded `index.html` and `error.html`.
4. Enabled **Static Website Hosting**.
5. Added a Bucket Policy for public read access.
6. Accessed the website using the S3 Website Endpoint.
